# CryptoApp

## Project setup
```
npm install
```

### Update settings
1. Create `src/utils/utils.js` file.
2. Get your API_KEY from https://min-api.cryptocompare.com.
3. Add following code to `utils.js`.

```javascript
const API_KEY =
  "YOUR_API_KEY";

export { API_KEY };

```

npm run serve
